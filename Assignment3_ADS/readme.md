## Description

This project compares the performance of two sorting algorithms and one search algorithm.
Implemented in Java, split across 5 classes.

## Algorithms in my code

Sorting:
- Bubble Sort
- Merge Sort

Search:
- Binary Search

## Files

- `Main.java` - entry point
- `ArrayGenerator.java` - generates and prints arrays
- `Sorter.java` - sorting algorithms
- `BinarySearch.java` - binary search
- `Experiment.java` - measures execution time

## What it does

The program generates random arrays, sorts them using both methods,
measures the time for each, and searches for an element using binary search.
Results are printed to the console for arrays of size 100, 1000, and 10000.
